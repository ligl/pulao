**如何判断当下那一刻是真强还是假强？**

* 市场现在是强还是弱？

* 这个突破是真还是假？

我的胜率将从随机变成**结构化优势**。


# I、实时供需强弱判断系统

整个系统分为**三层、九因子**，用来判断“**当下那一时刻市场究竟是强还是弱**”。

**总原则**

在任何关键位置，我只需做一件事：

**同时观察：**

**K线 → 成交量 → 持仓量 → 走势结构 → 多周期**

**这五种力量是否一致？**

如果一致，胜率高。

如果矛盾，胜率低。

如果完全矛盾，必假突破。


# 第一层：即时K线强弱（Momentum Micro-Structure）

这是当下那一刻的力量可视化。

## 因子1：拒绝 vs 接受（Rejection vs Acceptance）

这是判断转折真实性的核心。

* **拒绝（强反转信号）**

   * 长影线、价格被推回、实体小但方向明确。

* **接受（强突破信号）**

   * 实体靠近突破方向收盘。

交易含义：

* 转折要看“拒绝”

* 突破要看“接受”

我过去输在这里：

我把影线、实体看成技术细节，但真正关键是：

**价格是被市场推回（拒绝）还是被市场接纳（接受）？**

一次看懂市场的瞬时意图。

## 因子2：推进效率（Advancement Efficiency）

短周期里，连续两三根K，观察：

* 推得快、退得慢 → 强

* 推的慢、退得快 → 弱

* 推得快、退得快 → 混乱

* 推不动、量放不动 → 即将释放方向

推动效率是我“缺乏盘感“的核心原因之一。

## 因子3：动能一致性（Momentum Consistency）

突破前3-5根K是否一致朝某个方向倾斜？

* 连续三根小阳 > 突破成功率远高于大小阳小阴乱跳的结构

动能一致，是趋势尾端的特征；

动能不一致，是假突破的前兆。

# 第二层：价量持仓三联动（Vol-OI Structure）

这是最关键的供需判断部分。

## 因子4：成交量支持（Volume Confirmation）

价格强弱必须由量验证。

* **量↑价↑**→ 真强

* **量↑价↓**→ 真弱

* **量↓价↑** → 假强（空头并未充分参与竞争）

* **量↓价↓**→ 假弱（多头并未充分参与竞争）

价格的涨跌必须由多空真实力量的投入，否则，价格状态不牢固。

## 因子5：持仓量性质（OI Nature）

持仓量不是涨跌，而是“增仓的意义”。

* **增仓上涨** → 多头主动进场

* **增仓下跌** → 空头主动进场

* **减仓上涨** → 空头回补

* **减仓下跌** → 多头离场

* **量增OI不动** → 换手

关键：

“换手”是强弱中最被忽略的力量。

换手意味着：

**有人离场，有人接力。**

**市场正在交角。**

**这是转折区最重要的信号。**

## 因子6：价量持仓共振程度（Vol + OI Alignment）

三个方向是否一致？

一致 → 大概率真突破

不一致 →  大概率假突破

完全反向 →  必假突破

我过去完全忽略了三者共振，这是我判断力弱的主因。


# 第三层：结构强弱（Trend-Swing Structure）

这是趋势体系交易者的核心武器。

## 因子7：当前Swing力度 vs 上一Swing力度

衡量：

* 时间（K线数）

* 空间（点数）

* 推进效率（强弱）

趋势反转必有三大特征：

1. 新的Swing走得更快

2. 回撤更弱

3. 推进的波动幅度比前一段大

若我只关注突破，忽略整个Swing的强弱，这会让我经常买在局部小反弹。

## 因子8：关键位的反应方式（Reaction at KeyZone）

关键位观察两件事：

1. **拒绝还是接受**

2. **强者是否强，弱者是否弱**

例如：

* 在趋势中，如果价格触碰EMA20后迅速拒绝并推走 → 真强趋势

* 在突破位，如果被吸收，冲不上去 → 假突破

我缺乏这种“关键位行为学分析”。

## 因子9：多周期结构冲突（Multi-Timeframe Alignment）

低周期突破是否被高周期压制？

例：

* 5m强势突破

* 15m在下降趋势

* 1h在阻力位

→ 5m的突破价值几乎为零。

我现在的模型因为在“反转区、突破区”做决策，如果没有多周期过滤，会自然被假信号淹没。


# II、实时判断的执行流程（盘中10秒扫描）

每一次入场前做这件事：

## 10秒强弱扫描

1. 3根K的行为统一吗？

2. 价格是被拒绝还是被接受？

3. 推进速度快还是慢？

4. 量是否支持当前方向？

5. OI是增仓、减仓还是换手？

6. 量+OI是否共振？

7. 当前Swing的力道比前一段强吗？

8. 关键位反应是强还是弱？

9. 多周期是否一致？

如果：

* ≥7项一致 → 强

* ≤4项一致 → 弱

* 3项以上矛盾 → 假突破概率极高


经过以上，我将第一次感受到一种状态：


**“我终于能看清盘面在干什么了。”**


# III、实时供需判断：模板化流程

可以在盘中逐秒/逐K执行的判断流程，像checklist一样。

## 步骤1：确定我现在处于哪种结构环境

1. **趋势中回踩**

2. **趋势加速后的第一次回调**

3. **震荡边缘的突破**

不同的环境看强弱时的重点不同：

* 趋势中 → 看承接（回踩是否弱）

* 转折处 → 看主动性（谁率先发力）

* 边缘突破 → 看延续（突破是否走得出去）

这一步本质是：明确接下来应该观察哪一类力量。

## 步骤2：判断当前K线是“推进”还是“承接”

我的能力弱点就在这里。


**推进：**

* 向趋势方向打出空间

* 实体 > 前几根平均实体

* 成交量增（趋势中）

* 持仓量增（趋势中），持平或轻微减（震荡突破中）


**承接：**

* 回踩时阴线力度弱（下跌中的多头承接）

* 回调中成交量低

* 回调不创新低/不加速

* 持仓量减少或持平


任何一个方向要强，必须满足：

**推进强 + 承接稳**

反之：

**推进弱 + 承接差 = 虚假力量**


## 步骤3：看关键位置的反应

关键位置包含：

* 波段高/低

* 趋势背靠带

* 前方突破位

* 昨高/昨低

* 大结构节点

* EMA20、EMA60

* 通道边缘

* 区间边缘


判断标准是：

**关键位的影线 + 实体行为 = 力量的真实暴露点。**


例如：

**关键位被吃掉时 → 真突破**

* 实体贯穿

* 贯穿过程中无长影线反压

* 贯穿时成交量明显放大

* 贯穿后有承接（盘整、停顿，但不快速回落）

**关键位被拒绝时 → 假突破**

* 长影线

* 量能集中在影线处

* 突破那一根K线的推进没有持续性

* 突破后下一根反向穿回


## 步骤4：用持仓量判断真假力量

**1）上涨 + 增仓：趋势真强，多头真买入**

这是最好的情况。

**2）上涨 + 减仓：空头主动离场，上涨容易衰竭**

上涨不是真的“买”，而是空头撤退。

若后续没有增仓承接，上涨经常走不远。

**3）下跌 + 增仓：空头真打 + 多头被迫止损**

强空。

**4）下跌 + 减仓：多头被动离场，空头不敢追**

下跌容易弱掉，之后往往震荡 → 反弹。


我的问题是：

**我把上涨减仓、下跌减仓全部当成“方向弱”，导致判断混乱。**

我要做的是：

* 看“减仓后下一次推进”的表现，

* 如果减仓推动后，**下一次增仓跟随，那就是真力量要来了。**


## 步骤5：看“第二脚”

几乎所有真正能走出大行情的，都有一个共同点：

**不是一次突破成功，而是第二次突破成功。**

也就是说：

* 第一次突破更多是“试探”

* 第二次突破才是“攻击”


我经常在第一次突破时进入，导致止损，之后不敢再进，我要：

**不要试图判断第一次突破，只要等待第二次突破是否具有真实力量。**

这样会大幅提高我的胜率。
